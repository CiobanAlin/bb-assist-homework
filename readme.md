### Prerequisites

  1. Node version 7+
  2. bower
  3. npm

### Usage of this project

  1. Install npm dependencies
  ```
  npm install
  ```

  2. Install bower dependencies
  ```
  bower install
  ```

  3. Install gulp-cli
  ```
  npm install gulp-cli
  ```

  4. Run gulp
  ```
  gulp
  ```
